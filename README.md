# 2D-Platformer-Game

Simple 2D Platformer Game

A minimalist 2D platformer built using HTML, CSS, and JavaScript. The goal is to navigate a simple cube through various obstacles to reach the finish line, saving your progress at checkpoints along the way.

**Key Features**

Responsive Movement: Smooth horizontal control for the player cube.

Classic Jump Physics: Includes basic gravity and ground detection for jumping.

Obstacles: Static hazards to challenge the player.

Checkpoints: Save the player's progress and respawn location.

Single-Screen Design: All game logic and rendering are contained within one file for simplicity.

**How to Play**

Left Arrow

Right Arrow

Spacebar

Objectives

Navigate the cube past all obstacles.

Reach the Finish Line (usually a different colored square) to complete the level.
